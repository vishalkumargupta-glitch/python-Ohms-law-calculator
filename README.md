# Python Ohm's Law Calculator ⚡

A simple interactive Ohm's law calculator built with Python - practice project for electrical engineering basics.

## 🚀 Features
- Calculate Voltage, Current, Resistance, and Power
- Three calculation modes based on known values
- User-friendly menu interface

## 💻 How to Use
1. Run the script: `ohms_law_calculator.py`
2. Choose what values you know (1, 2, or 3)
3. Enter the values
4. Get instant results!
